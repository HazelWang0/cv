Ji. Wang, Sen. Wang, Zhi. Jiang, Zhi. Xie, Men. Li,
Presented is a novel approach to generate three-dimensional virtual human avatars driven by zero-shot text descriptions, based on a diffusion model. Utilizing this method, users can create lifelike, high-quality, and vivid virtual avatars under given textual prompts, eliminating the need for any reference photographs.
